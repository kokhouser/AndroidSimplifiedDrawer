Android Simplified Drawer

An Android library which provides a simple activity class to extend from to enable both left and right navigation drawers.

Download this project as a .zip file Download this project as a tar.gz file

Introduction

At its core, ASD provides two classes that (hopefully) make creating drawers easier - DrawerActivity and DrawerManager.

It is currently NOT ready for basic use.

A sample app is made available in the "app2" folder. Simply import it into your Android Studio and build. (Play Store listing coming soon).

DrawerActivity

DrawerActivity extends from ActionBarActivity (therefore an AppCompat theme should be used or extended), to ensure compatibility with devices down to API level 7 (They deserve all the love they can get).

DrawerActivity, as a class, contains three views that you'll need to be concerned about: mDrawerLayout, leftDrawerList, and rightDrawerList, all three must be included in your activity's layout files (More on that in the "Sample Code" section).

mDrawerLayout is Google's implementation of a layout that utilizes drawers. leftDrawerList and rightDrawerList are both, predictably, ListViews that make up your drawers. Both are optional, but the fact that you're considering using this library probably means you should use at least one of them. In the future, I hope to allow users to customize the views they want to use in the left and right drawers, instead of limiting them to the traditional ListViews.

Sample Code

Utilizing this library is (hopefully) rather simple, though it is currently lacking in functionality beyond creating extremely simple drawers. First, your activity layout files should contain the appropriate views.

Sample activity_main.xml:

<?xml version="1.0" encoding="utf-8"?>
<android.support.v4.widget.DrawerLayout
    xmlns:android="http://schemas.android.com/apk/res/android"
    android:id="@+id/drawer_layout"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    xmlns:tools="http://schemas.android.com/tools">

    <!-- MAIN CONTENT VIEW -->
    <RelativeLayout
        android:layout_width="fill_parent"
        android:layout_height="fill_parent"
        android:id="@+id/root_view">
    </RelativeLayout>

    <!-- MAIN CONTENT VIEW ENDS -->

    <!-- LEFT DRAWER
         Currently, it doesn't matter what you name your listViews, 
         but I may fix that in the future to reduce number of
         code necessary to define in the Java files. -->

    <ListView android:id="@+id/left_drawer"
        android:layout_width="200dp"
        android:layout_height="match_parent"
        android:layout_gravity="start"
        android:choiceMode="singleChoice"
        android:background="#ffffffff" />

    <!-- RIGHT DRAWER -->

    <ListView android:id="@+id/right_drawer"
        android:layout_width="200dp"
        android:layout_height="match_parent"
        android:layout_gravity="end"
        android:choiceMode="singleChoice"
        android:background="#ffffffff" />
</android.support.v4.widget.DrawerLayout>

Next, in your activity Java file, you should:

  • Extend from DrawerActivity
  • Set values of mDrawerLayout, leftDrawerList and rightDrawerList.
  • Create String array containing drawer item names.
  • Create int array containing resource ID's of icons.
  • Implement OnItemClickListener for drawers.
  • Create DrawerManager and pass required info to it.

Sample MainActivity.java

import android.os.Bundle;
import android.support.v4.widget.DrawerLayout;
import android.view.View;
import android.widget.AdapterView;
import android.widget.ListView;
import android.widget.Toast;

import com.kokhouser.simplifieddrawer.library.DrawerActivity;
import com.kokhouser.simplifieddrawer.library.DrawerManager;


public class MainActivity extends DrawerActivity {

    Toast toast;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
        toast = Toast.makeText(getApplicationContext(), null, Toast.LENGTH_SHORT);
        mDrawerLayout = (DrawerLayout) findViewById(R.id.drawer_layout);
        leftDrawerList = (ListView) findViewById(R.id.left_drawer);
        rightDrawerList = (ListView) findViewById(R.id.right_drawer);
        final String[] itemNames = new String[]{"Home", "Favourites", "Settings"};
        int[] iconNames = new int[]{R.drawable.dashboard_icon, R.drawable.favourite, R.drawable.settings_icon };
        class DrawerItemListener implements ListView.OnItemClickListener{
            @Override
            public void onItemClick(AdapterView<?> parent, View view, int position, long id) {
                switch (position) {
                    //Add outcomes of pressing nav drawer items
                    case 0:
                        toast.setText("Tapped on " + itemNames[0]);
                        toast.show();
                        break;
                    case 1:
                        toast.setText("Tapped on " + itemNames[1]);
                        toast.show();
                        break;
                    case 2:
                        toast.setText("Tapped on " + itemNames[2]);
                        toast.show();
                        break;

                }
            }
        }
        DrawerManager drawerManager = new DrawerManager();
        drawerManager.setlDrawerTitle("Navigation");
        drawerManager.setrDrawerTitle("Notifications");
        drawerManager.setMenuTitles(itemNames);
        drawerManager.setMenuIcons(iconNames);
        drawerManager.initializeLeftDrawer(this, new DrawerItemListener());
    }
}
